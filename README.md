# ☕ Lume - Café Experience

O **Lume** é um site moderno e responsivo desenvolvida para uma plataforma de pedidos de café. 
O projeto foca em uma estética minimalista, utilizando tons terrosos e uma tipografia limpa para proporcionar uma experiência de usuário acolhedora e funcional.

---

## 🛠️ Tecnologias Utilizadas

As principais ferramentas e tecnologias empregadas no desenvolvimento deste projeto foram:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada com foco em:
  - **Flexbox**: Para alinhamento e empilhamento dinâmico de elementos.
  - **Posicionamento Absoluto**: Para elementos decorativos de background.
  - **Cores RGBA/Hex8**: Para controle de opacidade e transparência de camadas.
- **Fontes Customizadas**: Implementação da família *Noto Sans Georgian* via `@font-face`.
- **JavaScript (ES6+)**: Implementação futura para lógica de autenticação e interatividade.
